# Codex Pulse icon prompt

Use case: logo-brand

Asset type: native macOS application icon for a real-time Codex quota monitor named Codex Pulse

Primary request: Create one distinctive premium app icon that combines a crisp white lightning bolt with a circular pulse / remaining-quota ring. The symbol should instantly suggest live monitoring, energy, and remaining capacity.

Scene/backdrop: the icon itself sits alone on a perfectly flat solid #ff00ff chroma-key background for later transparency removal. The background must be one uniform color with no shadows, gradients, texture, reflections, floor plane, or lighting variation.

Subject: a single centered macOS-style rounded-square icon tile, deep midnight navy to cobalt blue surface, with a bold white lightning bolt cutting through a bright electric-blue circular progress ring. Keep the silhouette compact and unmistakable at 16 px.

Style/medium: polished native macOS app icon, minimal geometric design, subtle layered depth and restrained glass-like highlights inside the icon tile, premium but not flashy.

Composition/framing: exact square canvas, centered icon tile with generous even padding around it; symmetrical visual weight; no elements outside the tile.

Color palette: midnight navy, cobalt blue, electric cyan, white. Do not use #ff00ff anywhere inside the icon.

Constraints: no text, no letters, no numbers, no watermark, no logos or trademarks, no mockup, no device, no cast shadow, no contact shadow, no reflection outside the icon. Crisp clean edges suitable for an app icon. Preserve a strong simple silhouette and balanced negative space.
